## Run Locally

Clone the project

```bash
https://github.com/msubham193/pixabay-clone-react
```

Go to the project directory

```bash
  cd pixabay-clone-react
```

Install dependencies

```bash
   npm install
```

Start the server

```bash
   npm start
```
